# Guía de Negociación Colaborativa

Guía interactiva para el juego de negociación colaborativa: **Cinco Proyectos, Dos Aprobaciones, Un Solo Macroproyecto**.

## Archivos

| Archivo | Descripción |
|---|---|
| `index.html` | Estructura de la página |
| `style.css` | Estilos y colores |
| `script.js` | Interactividad (acordeones) |

## Cómo publicar en GitHub Pages

1. Crea un repositorio en GitHub (puede ser público o privado con plan Pro).
2. Sube los tres archivos: `index.html`, `style.css` y `script.js`.
3. Ve a **Settings → Pages**.
4. En **Branch**, selecciona `main` y carpeta `/ (root)`.
5. Haz clic en **Save**.
6. En unos segundos tendrás tu guía en:  
   `https://<tu-usuario>.github.io/<nombre-del-repositorio>/`
